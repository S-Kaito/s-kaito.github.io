Hello, numpy!

Numpyの関数・メソッド一覧(np編)




A
arange
array arange(ndarray)
numpyの配列(ndarray)をPythonのarrayに変換する

パラメータ:
ndarray - numpy配列
返り値:
array - Python配列
argmax
n argmax(array,axis=None)
numpy配列の最大値の番地を取得する

パラメータ:
array - numpy配列
返り値:
n - 最大値がある番地
(axisを指定した場合はintではなくndarray)
array
ndarray array(array)
Python配列をnumpyの配列(ndarray)に変換する

パラメータ:
array - Python配列
返り値:
ndarray - numpy配列
D
dot
ndarray dot(ndarray1,ndarray2)
行列積の計算をする

パラメータ:
ndarray1 - numpy配列
ndarray2 - numpy配列
返り値:
ndarray - 計算済みのnumpy配列
F
full
ndarray full(shape,value)
valueで初期化されたnumpy配列を生成する

パラメータ:
shape - intなら長さshapeのnumpy配列
tupleなら形shapeのnumpy配列
value - 初期値 返り値:
ndarray - 初期化されたnumpy配列
M
max
n max(array,axis=None)
numpy配列の最大値を取得する

パラメータ:
array - numpy配列
返り値:
n - 最大値
(axisを指定した場合はintではなくndarray)
mean
n mean(array,axis=None)
numpy配列の平均値を取得する

パラメータ:
array - numpy配列
返り値:
n - 平均値
(axisを指定した場合はintではなくndarray)
min
n min(array,axis=None)
numpy配列の最小値を取得する

パラメータ:
array - numpy配列
返り値:
n - 最小値
(axisを指定した場合はintではなくndarray)
N
nanmax
n nanmax(array,axis=None)
numpy配列の最大値をNaNを無視して取得する

パラメータ:
array - numpy配列
返り値:
n - 最大値
(axisを指定した場合はintではなくndarray)
nanmin
n nanmin(array,axis=None)
numpy配列の最小値をNaNを無視して取得する

パラメータ:
array - numpy配列
返り値:
n - 最小値
(axisを指定した場合はintではなくndarray)
O
ones
ndarray one(shape)
1で初期化されたnumpy配列を生成する

パラメータ:
shape - intなら長さshapeのnumpy配列
tupleなら形shapeのnumpy配列
返り値:
ndarray - 初期化されたnumpy配列
W
where
ndarray where(conditions,x,y)
条件に合うかどうかで値が変わっているnumpy配列を生成する

パラメータ:
conditions - 条件文
x - 条件がTrueの所に代入される値
y - 条件がFalseの所に代入される値
返り値:
ndarray - 初期化されたnumpy配列
Z
zeros
ndarray zeros(shape)
0で初期化されたnumpy配列を生成する

パラメータ:
shape - intなら長さshapeのnumpy配列
tupleなら形shapeのnumpy配列
返り値:
ndarray - 初期化されたnumpy配列
